# Book API

API REST para gerenciamento de livros.

## Tecnologias

- Java 21
- Spring Boot
- Spring Data JPA
- H2 Database
- Spring Validation
- Swagger / OpenAPI
- JUnit 5
- Mockito

## Funcionalidades

- Criar livro
- Buscar todos os livros
- Atualizar livro
- Excluir livro
- Validação de dados

## Links apos execução do projeto

- Swagger - http://localhost:8080/swagger-ui/index.html#/
- Banco H2 - http://localhost:8080/h2-console/
